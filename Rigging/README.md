# Tarefa: Pipeline de Geração de Personagem Animado por Esqueleto 

## Autora
- Brenda Almeida

## Pré requisitos e dependências

- Compatível com Windows.
- Ter o git na máquina

## Instalação 

Clone o projeto para sua máquina,logo, após clonar você pode executá-lo a partir da pasta Build e clicando no executável ou selecionando a pasta e abrindo o projeto via UnityHub

## Uso

Para usar o projeto e ver as diferentes animações do personagem, basta clicar nos botões 1 2 e 3 do teclado.
Clique ESC para sair
